    UMA ESTRUTURA DO TIPO LISTA É UMA SEQUENCIA DE ELEMENTOS DO MESMO TIPO SEUS ELEMENTOS POSSUEM
    ESTRUTURA INTERNA ABSTRATA OU SEJA, SUA COMPLEXIDADE É ARBITRARIA E NÃO AFETA O SEU
    FUNCIONAMENTO
    
    
    
    UMA LISTA PODE POSSUIR N (N >= 0) ELEMENTOS
    APLICAÇÕES:
        CADASTRO DE FUNCIONARIOS
        ITENS EM ESTOQUE
        CARTAS DE BARALHO ...
    
    EM UMA LISTA PODEMOS REALIZAR AS SEGUINTES OPERAÇÕES BASICAS
        CRIAÇÃR DA LISTA
        INSERSAO DA LISTA 
        EXCLUSAO DE UM ELEMENTO
        INCLUSAO DE UM ELEMENTO
        ACESSO A UM ELEMENTO
        DESTRUICAO DE UM ELEMENTO
        
        
    ESSAS OPERAÇÕES DEPENDEM DO TIPO DE ALOCAÇÃO DE MEMORIA USADA
        ESTATICA
        DINAMICA
        
    ALOCAÇÃO ESTÁTICA   
        O ESPAÇO MEMORIA LIMITADO É ALOCADA NO MOMENTO DA COMPILAÇÃO 
        EXIGE A DEFINICAO DO NÚMERO MÁXIMO DE ELEMENTOS DA LISTA
        ACESSO SEQUENCIA : ELEMENTOS CONSECUTIVOS NA MEMORIA

    ALOCAÇÃO DINAMICA
        O ESPAÇO DE MEMORIA É COLOCADO EM TEMPO DE EXECUÇÃO
            A LISTA CRESCE A MEDIDA QUE NOVOS ELEMENTOS SÃO ARMAZENADOS, E DIMINUI A 
            MEDIDA QUE ELEMENTOS SSÃO REMOVIDOS 
        O ACESSO ENCADEADO : CADA ELEMENTO PODE ESTAR EM UMA DISTANCIA DA MEMÓRIA. PARA
        ACESSAR UM ELEMENTO É PRECISO PERCORRER TODOS OS SEUS ANTECESSORES NA LISTA


========
	LISTAS SEQUENCIAL ESTÁTICAS OU LISTA LINEAR ESTÁTICA
	É UM TIPO DE LISTA ONDE O SUCESSOR DE UM ELEMENTO OCUPA A POSICAO FISICA SEGUINTE DO MESMO

	VANTAGENS : ACESSO RAPIDO E DIRETO AOS ELEMENTOS (INDICES)
		    ELEMENTO CONSTANTE PARA ACESSAR UM ELEMENTO
	 	    FACILIDADE EM MODIFICAR INFROMAÇOES
	
    
	DESVANTAGENS DO USO DE ARRY : ESTAMOS TRABALHANDO COM TAMANHO FIXO
		DEFICULDADE PARA INSERIR E EXCUIR UM ELEMENTO ENTRE OS OUTROS DOIS : É NECESSARIO DESLOCAR OS ELEMENTOS
